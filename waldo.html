Where's Waldo?

Open Ended Project 3 - Hailey Schoppe - CS2300

Welcome to Where's Waldo, a game about spotting Waldo in as few guesses as possible!

Upload an image file that includes Waldo hidden somewhere within it. If Waldo is not found, the game will default to an image that we have.

The game will then locate Waldo and, based on where you click, tell you how far away you are!
